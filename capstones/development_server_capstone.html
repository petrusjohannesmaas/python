This articles contains all the steps you need to take to configure development server that you can access on Visual Studio Code (locally and remotely). You must be running Windows 10 (Build 19041 and higher) or Windows 11 to use WSL and run the commands below.

ðŸ”µ Run your PowerShell as administrator and install WSL on your computer :

wsl --install

# By default, the installed Linux distribution will be Ubuntu. This can be changed using the `-d` flag.
wsl --install -d Debian

#(Optional:) unregister any other distributions so you're only working on Debian
wsl --unregister <Distribution Name>

ðŸ”µ Open the Debian console in your VScode terminal and run these commands:

# update the Debian system
sudo apt update
sudo apt upgrade

# Install Git and Curl
sudo apt install git
sudo apt install curl

# Download Homebrew:
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Once the install is done make sure Brew is in PATH by running:
cat ~/.bashrc

# run this to update PATHs
source ~/.bashrc 

# Output should be "Your system is ready to brew" if you run this command:
brew doctor 

#See available Homebrew versions:
brew search python 

#Install the latest/highest version
brew install python@3.xx

#Confirm python installation
python3 -V

ðŸ“¦ If you want to manage your packages and versions later:

brew --help
brew update # Update internal list of formulae
brew brew upgrade <package> #Update a specific formula

ðŸ”µ Configure your version control:

#Add your code folder (with version control)
mkdir code && cd code && git init

ðŸ”µ In your VScode Command Pallet do the following:

  • Press f1 to open the command pallet
  • Type Python:Select Interpreter select the Homebrew interpreter path.
  • Create: New File and select Python File
  • Add print("Hello Debian!") to the file and save it in the Python repository as test.py
  • Type Python: Run Python File in Terminal (or just press the run button) to see the code in your terminal

ðŸ”„ Sync your VScode settings with your Github account:

  • Commit and sync your changes on the main branch of your /python repository

  • Enable Settings Sync to preconfigure other VScode terminals automatically

  • Enable Remote Tunnel Access for remote access to the Debian server
